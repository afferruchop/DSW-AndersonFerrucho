Proyecto de sitio web para el curso Diseño de Sitios Web

Proposito

Este proyecto se elabora con la intención de cumplir con el requerimiento solicitado por el instructor del curso, el cual establecio los lineamientos del desarrollo en la guia de actividades, al seguir la orientación del instructor, se espera cumplir con el objetivo de la evaluación y adquirir las destrezas planteadas en el silabus.

Como contenido la guía solicita hacer una investigación sobre los conceptos planteados en la introducción, los cuales serviran como soporte base para mostrar un material con contenido coherente sobre el tema.

Introducción

Las temáticas propuestas para el desarrollo del contenido son las siguientes:

  • Etiquetas semanticas en HTML
  • Listas en HTML
  • Enlaces HTML
